## 开发前须知
1. 全局变量：Logger
1. 安装全局：node-dev
1. 运行 `npm install`

## 开发模式启动
```bash
npm run dev
```

## 打包
```bash
npm run build   # 使用 webpack 进行打包
```

## 模拟生产环境启动
```bash
npm run stage
```

## 部署
```bash
npm run deploy
```
